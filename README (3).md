# AC-to-DC-Converter



## 📝 Description

The AC-to-DC-Converter project is a robust implementation of a power supply unit designed to efficiently transform high-voltage alternating current into a stable, low-voltage direct current output. This project showcases the fundamental principles of power electronics, incorporating critical stages such as full-wave bridge rectification, capacitive filtering to reduce ripple voltage, and precise voltage regulation. Engineered for reliability and performance, it serves as an excellent educational resource for students and a practical building block for hobbyists looking to power sensitive electronic components and prototypes with a clean DC signal.
